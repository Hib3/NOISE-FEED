# NOISE FEED design reference

`ideas.zip` から抽出した参照画像です。

UIにはこの画像をそのまま背景として貼らず、以下の特徴を抽出して反映します。

- 古新聞の段組みと高密度な短冊
- パンクZINEの切り貼り、斜め配置、黒ベタ
- 破れ紙の不規則な縁と紙片の影
- くしゃくしゃ紙、コピー機ノイズ、網点
- 赤茶色のテープと検版スタンプ

追加生成素材:

- `apps/pwa/public/assets/generated/punk-collage.jpg`

生成プロンプト要旨:

> Gritty black-and-white punk zine collage sheet made from torn newspaper strips, photocopied halftone photo fragments, ragged paper edges, pasted labels, rough black blocks, and small red-brown tape marks. No readable real words, no logos, no watermark.
